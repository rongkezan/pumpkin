时间/空间复杂度

1. 时间复杂度

常对幂指阶：常数级<对数级<幂数级<指数级<阶乘

结论1：顺序执行的代码只影响常数项，可以忽略

结论2：只需挑循环中的一个基本操作分析它的执行次数与n的关系即可

练习：计算时间复杂度

设总循环的次数为x，则由循环条件可知，循环结束刚好满足2^x > n

x = log₂n + 1

T(n) = O(log₂n)

2. 空间复杂度

递归调用空间复杂度：空间复杂度 = 递归调用的深度
