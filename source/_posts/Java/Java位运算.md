左移：低位补0

```
5 << 2 = 101 -> 10100 = 20
```

右移：高位补0

```
5 >> 2 = 101 -> 0001 = 1
```

位与：两边同时为1才为1，否则为0

```
5 & 3 = 101 & 011 = 001 = 1
4 & 1 = 100 & 001 = 000 = 0
5 & 12 = 0101 & 1100 = 0100 = 4
```

位或：两边只要有一个为1就为1，否则为0

```
5 | 3 = 101 | 011 = 111 = 7
5 | 1 = 101 | 001 = 101 = 5
```

位异或：不同为1，相同为0

```
11^7 = 1011 ^ 0111 = 1100 = 12
```
